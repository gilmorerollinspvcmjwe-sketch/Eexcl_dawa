# Excel Aim Trainer 游戏增强计划

## 一、新手引导系统

### 1.1 首次进入操作教程
**触发条件**：玩家首次打开游戏

**教程步骤**：
1. **欢迎界面**
   - 显示："欢迎来到 Excel Aim Trainer"
   - 说明：这是一个伪装成Excel的练枪游戏
   - 按钮："开始教程" / "跳过"

2. **基础操作教学（3步）**
   - 第1步：点击目标
     - 高亮显示一个目标
     - 文字提示："点击目标进行射击"
     - 玩家点击后进入下一步
   
   - 第2步：移动鼠标
     - 文字提示："移动鼠标瞄准，准星会跟随"
     - 演示准星跟随效果
   
   - 第3步：紧急隐藏
     - 文字提示："按 Esc 或点击 ─ 按钮隐藏游戏"
     - 演示隐藏/恢复效果

3. **完成教程**
   - 显示："教程完成！准备好开始训练了吗？"
   - 按钮："立即开始" / "返回主界面"

**技术实现**：
- 使用 localStorage 记录 `hasCompletedTutorial` 标志
- 创建 `TutorialOverlay` 组件
- 高亮效果使用 CSS `box-shadow` 和 `animation`

---

### 1.2 各模式首次玩法说明
**触发条件**：玩家首次进入某个游戏模式

**显示方式**：
- 游戏开始前显示半透明遮罩
- 左侧显示玩法说明文字
- 右侧显示示意图（可选）

|------|----------|

| 无限模式 | "没有时间和生命限制，适合放松练习，失误3次游戏结束" |
| 禅模式 | "无压力练习，没有时间限制，专注于精准度" |
| 爆头线模式 | "所有目标都在同一高度，练习爆头肌肉记忆" |
| 移动射击 | "目标会移动，需要预判位置进行射击" |

| 目标切换 | "多个目标同时出现，练习快速切换瞄准" |
| 反应测试 | "测试你的反应速度，目标出现后尽快点击" |

**技术实现**：
- 使用 localStorage 记录各模式的 `hasSeenModeTutorial[modeId]`

### 1.3 练习模式
**模式特点**：
- 无时间限制
- 无生命限制
- 无得分压力
- 专门用于熟悉操作

**练习内容**：
1. **基础点击练习**
   - 静态目标，熟悉点击操作
   
2. **移动目标练习**
   - 慢速移动目标，练习跟枪
   
3. **爆头练习**
   - 只有头部目标，练习精准度
   
4. **反应练习**
   - 目标随机出现，练习反应速度

**UI位置**：
- 在 Sheet1 的"经典模式"区域添加"练习模式"按钮
- 图标：📚
- 描述：「无压力练习，熟悉操作」

## 二、新游戏模式

### 2.1 生存模式
**核心机制**：
- 玩家有3点生命值
- 击中目标 +1 分
- 失误（未击中）-1 生命值
- 生命值为0时游戏结束
- 随着时间推移，目标生成速度加快

**UI显示**：
- 在 HUD 显示 ❤️❤️❤️ 生命值
- 失误时生命值图标闪烁变红
- 游戏结束显示生存时间和得分

**难度递增**：
- 每30秒，目标生成速度 +10%
- 每60秒，目标持续时间 -10%

---

### 2.2 仅头部模式
**核心机制**：
- 只有击中头部算分
- 击中其他部位显示"无效"提示
- 头部目标更小，移动更快

**UI显示**：
- 目标只有头部显示（身体透明或淡化）
- 击中非头部时显示 💨 无效提示
- 得分只计算爆头

**适用场景**：
- 专门练习爆头精准度
- 适合有一定基础的玩家

---

### 2.3 无畏契约技能模式（Valorant Style）
**核心概念**：
模拟无畏契约中的技能对抗场景，玩家需要在复杂环境下击中敌人。

**技能系统**：

#### 1. 闪光弹（Flash）
**效果**：
- 屏幕中央出现闪光弹图标
- 1.5秒后爆炸
- 爆炸时屏幕变白 0.5秒
- 如果玩家在爆炸前击中闪光弹，可抵消效果

**UI展示**：
```
┌─────────────────────────────────┐
│         ⚡ 闪光弹 1.5s           │  ← 顶部倒计时
│                                 │
│              💣                 │  ← 中央闪光弹图标
│                                 │
└─────────────────────────────────┘
```

#### 2. 烟雾弹（Smoke）
**效果**：
- 屏幕某区域出现灰色烟雾
- 烟雾中目标半透明
- 持续5秒

**UI展示**：
- 灰色半透明遮罩覆盖部分区域
- 烟雾边缘有渐变效果

#### 3. 震撼弹（Concussion）
**效果**：
- 屏幕边缘出现红色警告
- 准星晃动 2秒
- 移动速度降低

**UI展示**：
- 屏幕边缘红色闪烁边框
- 准星添加随机偏移

#### 4. 治疗/护盾（Heal/Shield）
**效果**：
- 绿色/蓝色道具随机出现
- 击中后恢复生命值/获得护盾

**UI展示**：
- 💚 治疗道具（绿色）
- 💙 护盾道具（蓝色）

#### 5. 敌方技能提示
**效果**：
- 敌人使用技能前，头顶显示技能图标
- 给玩家 0.5-1秒 反应时间

**UI展示**：
```
        ⚡  ← 敌人头顶显示即将使用的技能
      [敌人]
```

**模式玩法**：
1. 基础阶段：普通目标 + 偶尔闪光弹
2. 进阶阶段：加入烟雾弹和震撼弹
3. 专家阶段：多种技能组合，敌人也会使用技能

**实现思路**：
- 创建 `SkillSystem` 类管理技能
- 技能作为特殊目标类型
- 使用状态机管理技能效果

## 四、击中反馈优化

### 4.1 Excel单元格选中效果
**灵感**：Excel选中单元格时的边框效果

**实现方案**：
```css
.hit-feedback {
  /* 绿色边框 */
  box-shadow: 0 0 0 2px #22c55e, 0 0 0 4px rgba(34, 197, 94, 0.3);
  
  /* 动画效果 */
  animation: excelCellSelect 0.3s ease-out;
}

@keyframes excelCellSelect {
  0% { 
    box-shadow: 0 0 0 0 transparent, 0 0 0 0 transparent;
    transform: scale(1);
  }
  50% { 
    box-shadow: 0 0 0 3px #22c55e, 0 0 0 6px rgba(34, 197, 94, 0.2);
    transform: scale(1.05);
  }
  100% { 
    box-shadow: 0 0 0 2px #22c55e, 0 0 0 4px rgba(34, 197, 94, 0.3);
    transform: scale(1);
  }
}
```

**效果描述**：
1. 击中时，目标周围出现绿色双层边框
2. 内层边框实线，外层边框虚线/半透明
3. 类似Excel选中单元格的视觉效果
4. 持续0.3秒后消失

### 4.2 连击反馈增强
**连击数显示**：
- 在准星附近显示当前连击数
- 10连击以上数字变大并闪烁
- 连击中断时显示"连击中断"提示

**视觉反馈层级**：
```
普通击中：绿色边框
5连击：   绿色边框 + 小数字
10连击：  金色边框 + 中等数字 + 轻微震动
20连击：  橙色边框 + 大数字 + 屏幕闪烁
30连击：  红色边框 + 超大数字 + 全屏特效
```
