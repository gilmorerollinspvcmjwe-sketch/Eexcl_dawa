## 🎮 这是什么？

一个**完美伪装成 Excel 表格**的练枪游戏！

### 为什么选它？

| 场景 | 传统练枪软件 | Excel Aim Trainer |
|------|-------------|-------------------|
| 上班时 | ❌ 太明显，容易被发现 | ✅ 看起来在认真工作 |
| 隐蔽性 | ❌ 游戏界面暴露 | ✅ 完美伪装成 Excel |
| 紧急时刻 | ❌ 来不及关闭 | ✅ Esc 一键隐藏 |
| 专业度 | ❌ 功能单一 | ✅ 多种训练模式 |

### 核心卖点

- 👔 **上班摸鱼神器** - 看起来在认真工作，实则在练枪
- 🎯 **专业练枪工具** - 多种训练模式，提升反应速度
- 🎨 **极致伪装** - 完整的 Excel 界面，行列标识、公式栏一应俱全
- ⚡ **紧急隐藏** - Esc 一键隐藏，悬停左上角 3 秒自动伪装
- 🎮 **游戏化体验** - 连击、成就、排行榜，让练习更有趣

## ✨ 核心特性

### 🕵️ 极致伪装系统

```
老板走过来看到了什么？
→ 一个普通的 Excel 表格
→ 员工在认真处理数据
→ 完全看不出是游戏！
```

- ✅ **完整 Excel 界面**：标题栏、工具栏、公式栏、行列标识 A-Z
- ✅ **工作表标签**：Sheet1/2/3/4 切换，像真的一样
- ✅ **紧急隐藏**：
  - `Esc` 键：瞬间隐藏游戏
  - `悬停左上角 3 秒`：自动伪装
  - `F5` 键：恢复游戏
- ✅ **伪装图片上传**：上传你的真实 Excel 截图，隐藏时显示

### 🎯 6 种训练模式

| 模式 | 说明 | 适合场景 | 难度 |
|------|------|----------|------|
| ⏱️ **限时模式** | 30/60/120 秒挑战最高分 | 快速热身 | ⭐⭐ |
| ♾️ **无限模式** | Miss 3 次结束，追求最高连击 | 极限挑战 | ⭐⭐⭐ |
| 🧘 **禅模式** | 无压力练习，目标不消失 | 放松练习 | ⭐ |
| 🎯 **爆头线模式** | 专项训练，只有爆头计分 | 精准训练 | ⭐⭐⭐⭐ |
| 🏃 **移动靶模式** | 目标持续移动 | 跟枪练习 | ⭐⭐⭐ |
| 👁️ **Peek 模式** | 目标快速闪现 | 反应训练 | ⭐⭐⭐⭐⭐ |

## 🏆 游戏机制详解

### 部位系统

```
    ●        ■        ▲
   头部      身体      脚部
   100分     50分      25分
   暴击！    命中      擦伤
```

| 部位 | 图标 | 分数 | 说明 |
|------|------|------|------|
| 头部 | ● | 100 分 | 暴击！伤害最高 |
| 身体 | ■ | 50 分 | 标准命中 |
| 脚部 | ▲ | 25 分 | 擦伤，分数最低 |

### 连击加成系统

保持连击，分数翻倍！

```
连击数    倍率    效果
─────────────────────────
  5      1.5x    🔥
 10      2.0x    🔥🔥
 20      2.5x    🔥🔥🔥
 30      3.0x    🔥🔥🔥🔥
```

### 命中特效

- 💯 **得分飘字**：命中时显示得分数字飘起
- 🎯 **爆头特效**：头部命中特殊动画
- 🔥 **连击提示**：连击达到阈值时屏幕提示
- ✨ **视觉反馈**：不同部位不同颜色反馈

## 🚀 快速开始

### 方式一：在线体验（推荐）

```bash
# 克隆项目
git clone https://github.com/gilmorerollinspvcmjwe-sketch/Eexcl_dawa.git

# 进入目录
cd Eexcl_dawa

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 浏览器访问 http://localhost:3460
```

### 方式二：构建生产版本

```bash
npm run build

# 生成的 dist 目录可以部署到任何静态服务器
```

### 系统要求

- Node.js 18+
- 现代浏览器（Chrome、Firefox、Edge、Safari）
- 推荐分辨率：1920x1080 或更高

## 🎮 操作指南

### 快捷键

| 按键 | 功能 | 使用场景 |
|------|------|----------|
| 🖱️ **鼠标左键** | 点击目标 | 正常游戏 |
| ⌨️ **Esc** | 隐藏/恢复游戏 | 紧急情况 |
| ⏸️ **P** | 暂停/继续 | 临时有事 |
| 🖱️ **悬停左上角** | 3秒后自动隐藏 | 无缝切换 |

### 工作表导航

点击底部标签切换：

- **📊 Sheet1 (游戏中心)** - 选择模式、查看教程、开始游戏
- **🎯 Sheet2 (训练场)** - 主游戏区域，点击目标射击
- **📈 Sheet3 (统计)** - 查看战绩、成就、历史趋势
- **⚙️ Sheet4 (设置)** - 调整准星、音效、单元格样式等

## 💡 高手技巧

### 初级技巧
1. **🎯 头部优先** - 头部 100 分，性价比最高
2. **👁️ 提前预判** - 目标出现前把鼠标移到预测位置
3. **📈 循序渐进** - 从普通难度开始，逐步提升

### 进阶技巧
4. **🔥 保持连击** - 连击倍率是高分关键，宁可慢不要断
5. **⚡ 快速切换** - 熟悉 Esc 和 F5，紧急情况秒切换
6. **🎨 自定义准星** - 找到适合自己的准星样式和大小

### 大师技巧
7. **🌈 利用单元格颜色** - 复杂颜色模式练习动态视力
8. **🏃 移动靶预判** - 计算目标移动轨迹，提前瞄准
9. **🎯 爆头线训练** - 专项练习头部瞄准，形成肌肉记忆
