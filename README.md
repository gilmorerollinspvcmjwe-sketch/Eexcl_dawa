# Excel Aim Trainer 🎯

一个伪装成 Excel 表格的练枪游戏，完美适合上班摸鱼练习反应速度！

## 特性

- **极致伪装**：完整的 Excel 界面模拟，包括标题栏、工具栏、公式栏、行列标识
- **三种游戏模式**：
  - 限时模式：30/60/120 秒挑战最高分
  - 无限模式：Miss 3 次结束，追求最高连击
  - 禅模式：无压力练习，目标不消失
  - 爆头线模式：专项训练，只有爆头计分
- **目标系统**：头部 (●/100 分)、身体 (■/50 分)、脚 (▲/25 分)
- **连击加成**：最高 3x 倍率
- **命中特效**：命中时显示得分飘字动画，爆头和连击有特殊效果
- **紧急隐藏**：Esc 键或鼠标悬停左上角 3 秒快速隐藏
- **暂停功能**：按 P 键暂停游戏，显示半透明遮罩和继续按钮
- **统计数据**：命中率、平均得分、成就系统、得分趋势图
- **自定义设置**：
  - 难度、目标频率、持续时间、目标大小
  - 7 种准星样式：十字、点、圆、T 形、瓦罗兰特、CS2、CF
  - 准星大小和颜色自定义，实时预览
  - 音效开关，安静模式不扰民
  - 灵敏度预设，模拟不同游戏手感
- **公式栏联动**：Name Box 实时显示选中单元格坐标

```bash

```

```bash

```

### 构建生产版本

```bash
npm run build
```

## 操作说明

| 操作 | 说明 |
|------|------|
| 鼠标点击 | 点击单元格中的目标 |
| Esc | 紧急隐藏游戏 |
| F5 | 恢复游戏 |
| P | 暂停/继续游戏 |
| 鼠标悬停左上角 3 秒 | 紧急隐藏 |
| 设置面板 | 切换至"设置"工作表，调整准星、音效等 |

## 游戏技巧

1. **头部优先**：头部目标分数最高（100 分），优先点击
2. **保持连击**：连击达到 5/10/20/30 时分别获得 1.5x/2x/2.5x/3x 加成
3. **预判位置**：熟悉 B2-AD50 区域的目标生成位置
4. **难度选择**：从普通难度开始，熟练后挑战专家难度
5. **合理使用暂停**：按 P 键暂停游戏，紧急情况不慌乱

## 开发

### 添加新功能

1. 在 `src/components/` 创建新组件
2. 在 `src/types.ts` 添加类型定义
3. 在 `src/hooks/useGameLogic.ts` 添加游戏逻辑
4. 在 `src/App.tsx` 中集成

### 调试技巧

- 打开浏览器开发者工具查看控制台
- 使用 React DevTools 检查组件状态
- 本地存储数据可在 Application → Local Storage 查看
