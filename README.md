## 🎮 这是什么？

一个**完美伪装成 Excel 表格**的练枪游戏！

- 👔 **上班摸鱼神器** - 看起来在认真工作，实则在练枪
- 🎯 **专业练枪工具** - 多种训练模式，提升反应速度
- 🎨 **极致伪装** - 完整的 Excel 界面，行列标识、公式栏一应俱全
- ⚡ **紧急隐藏** - Esc 一键隐藏，悬停左上角 3 秒自动伪装

## ✨ 核心特性

### 🕵️ 极致伪装系统
- ✅ 完整的 Excel 界面模拟（标题栏、工具栏、公式栏、行列标识）
- ✅ **紧急隐藏**：Esc 键或鼠标悬停左上角 3 秒
- ✅ **暂停功能**：P 键暂停，显示半透明遮罩
- ✅ 伪装图片上传：上传真实 Excel 截图，隐藏时显示

### 🎯 多种训练模式

| 模式 | 说明 | 适合场景 |
|------|------|----------|
| ⏱️ **限时模式** | 30/60/120 秒挑战最高分 | 快速热身 |
| ♾️ **无限模式** | Miss 3 次结束，追求最高连击 | 极限挑战 |
| 🧘 **禅模式** | 无压力练习，目标不消失 | 放松练习 |
| 🎯 **爆头线模式** | 专项训练，只有爆头计分 | 精准训练 |
| 🏃 **移动靶模式** | 目标持续移动 | 跟枪练习 |
| 👁️ **Peek 模式** | 目标快速闪现 | 反应训练 |

### 🎨 深度自定义

#### 准星系统
- 11 种准星样式：十字、点、圆、T 形、瓦罗兰特、CS2、CF、Apex、守望先锋等
- 准星大小、颜色自定义
- 实时预览效果

#### 单元格设置（新增！）
- 单元格大小调节（影响难度）
- 6 种颜色模式：默认、随机、密度、棋盘、热图、彩虹
- 颜色强度调节
- 动态闪烁效果

#### 游戏设置
- 6 档难度：从"非常简单"到"专家"
- 目标生成频率、持续时间、大小
- 音效开关（安静模式）
- 灵敏度预设（模拟不同游戏手感）

### 🏆 游戏机制

#### 部位系统
| 部位 | 图标 | 分数 |
|------|------|------|
| 头部 | ● | 100 分 |
| 身体 | ■ | 50 分 |
| 脚部 | ▲ | 25 分 |

#### 连击加成
- 5 连击：1.5x 倍率
- 10 连击：2x 倍率
- 20 连击：2.5x 倍率
- 30 连击：3x 倍率

#### 命中特效
- 得分飘字动画
- 爆头特殊效果
- 连击视觉反馈

### 📊 统计系统
- 命中率、平均得分追踪
- 历史趋势图表
- 成就系统
- 个人最佳记录

## 🚀 快速开始

### 在线体验

```bash
# 克隆项目
git clone <repository-url>

# 进入目录
cd excel-aim-trainer

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问 http://localhost:3460
```

### 构建生产版本

```bash
npm run build
```

## 🎮 操作指南

| 操作 | 说明 |
|------|------|
| 🖱️ **鼠标点击** | 点击单元格中的目标 |
| ⌨️ **Esc** | 紧急隐藏/恢复游戏 |
| ⏸️ **P** | 暂停/继续游戏 |
| 🖱️ **悬停左上角 3 秒** | 快速隐藏 |
| 📊 **Sheet 切换** | 点击底部标签切换工作表 |

### 工作表说明

- **📊 Sheet1 (游戏中心)** - 选择模式、开始游戏
- **🎯 Sheet2 (训练场)** - 主游戏区域
- **📈 Sheet3 (统计)** - 查看战绩、成就
- **⚙️ Sheet4 (设置)** - 调整所有游戏参数

## 💡 游戏技巧

1. **🎯 头部优先** - 头部目标分数最高（100 分），优先点击
2. **🔥 保持连击** - 连击达到阈值获得倍率加成，分数翻倍
3. **👁️ 预判位置** - 熟悉目标生成区域，提前准备
4. **📈 循序渐进** - 从普通难度开始，熟练后挑战专家
5. **⏸️ 善用暂停** - 紧急情况按 P 键暂停

## 🎨 特色功能详解

### 单元格颜色系统
6 种独特的单元格颜色模式，增加游戏难度和视觉体验：
- **默认** - Excel 经典风格
- **随机** - 完全随机颜色，高难度
- **密度** - 区域密度变化
- **棋盘** - 棋盘格模式
- **热图** - 中心亮边缘暗
- **彩虹** - 彩虹渐变效果

### 视觉系统
- 敌人颜色和谐模式（互补色、类似色、三色组等）
- 出现动画（淡入、弹出、滑入、弹跳、闪烁）
- 敌人文字样式自定义

### FPS 训练模式
专为 FPS 玩家设计的训练模式：
- 移动靶训练
- Peek 反应训练
- 爆头线专项训练
