# Excel Aim Trainer - FPS 训练模式 UI 设计文档

## 概述

本文档定义了 Excel Aim Trainer 的 FPS 训练模式 UI 设计规范，包含训练模式选择、配置面板、训练 HUD、训练报告界面以及多格敌人视觉设计。

## 5. 多格敌人视觉设计

       ╱ ╲
    手 (L)  身体 (body)  手 (R)
      ●     ╱═╲═╲    ●

### 5.2 部位颜色方案

| 部位 | 颜色 (HEX) | 显示文字 | 分值 |
|------|------------|----------|------|
| 头部 | #dc2626 (红) | 头 | 150 |
| 身体 | #f97316 (橙) | 身 | 100 |
| 左手 | #eab308 (黄) | 手 | 60 |
| 右手 | #eab308 (黄) | 手 | 60 |
| 脚 | #6b7280 (灰) | 脚 | 40 |

### 5.3 部位 HP 条显示

```
┌──┐
│▓▓│ ← 满血 (绿色 #22c55e)
└──┘

┌──┐
│░░│ ← 受伤 (黄色 #eab308)
└──┘

┌──┐
│▓░│ ← 危急 (红色 #ef4444, 闪烁)
└──┘

┌──┐
│××│ ← 破坏 (灰色 #9ca3af, 穿透显示)
└──┘
```

**HP 条规范:**
- 宽度: 单元格宽度的 80%
- 高度: 3px
- 位置: 部位文字上方 2px
- 背景: rgba(0,0,0,0.3)
- 动画: 血量变化时 200ms 平滑过渡

### 5.4 优先级图标

| 优先级 | 图标 | 颜色 | 出现条件 |
|--------|------|------|----------|
| 紧急 | 🔴 | #dc2626 | 目标即将消失 |
| 高 | 🟠 | #f97316 | 时间 < 2.5s |
| 中 | 🟡 | #eab308 | 时间 < 4s |
| 低 | 🟢 | #22c55e | 默认 |

**优先级动画:**
- 紧急: 脉冲闪烁 (300ms 间隔)
- 高: 缓慢闪烁 (600ms 间隔)
- 中/低: 静态显示

### 5.5 受伤/破坏状态效果

| 状态 | 效果描述 |
|------|----------|
| 正常 | 正常显示部位文字和颜色 |
| 受伤 | 部位文字闪烁黄色, 播放 hit 音效 |
| 危急 | 部位文字闪烁红色, 添加脉冲动画 |
| 破坏 | 显示 "✕", 穿透显示, 播放 destroy 音效 |

**受伤动画:**
- 持续时间: 150ms
- 效果: 文字短暂放大 (scale: 1.2) + 颜色变化
- 缓动: ease-out

### 5.6 移动动画效果

| 移动模式 | 动画描述 |
|----------|----------|
| 直线移动 | 沿水平/垂直方向匀速移动, 速度可配置 |
| 正弦波 | 沿路径做正弦摆动, 摆动幅度 2 格 |
| 弹跳 | 在边界处反弹, 类似乒乓球轨迹 |
| 随机 | 随机选择方向移动, 每 1-2 秒改变方向 |

**移动动画参数:**
- 平滑移动: 使用 CSS transform, 60fps
- 边界处理: 到达边界后反弹或消失
- 速度: slow=0.5格/s, normal=1格/s, fast=2格/s, extreme=3格/s

### 5.7 探头动画效果

```
隐藏状态 → 探头中 (0-50%) → 完全可见 (50-100%) → 停留 → 返回
```

| 阶段 | 进度 | 效果 |
|------|------|------|
| 探头中 | 0-50% | 从掩体边缘平滑滑出, scale 从 0.8 到 1.0 |
| 完全可见 | 50% | 目标完全展示, 可以射击 |
| 停留 | 50-90% | 保持可见状态 |
| 返回 | 90-100% | 平滑收回掩体后方 |

**探头动画参数:**
- 探头时长: 可配置 (300ms - 2000ms)
- 缓动函数: ease-in-out
- 动画类型: transform + opacity 组合
