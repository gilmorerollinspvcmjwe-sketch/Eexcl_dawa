# 经典模块玩法补充与执行总计划

- 日期：2026-04-15
- 范围：祖玛 / 三消 / 吃豆人
- 目标：把当前三个“可玩骨架”模块推进到“核心玩法完整、内容层成型、长线结构可持续扩展”的状态
- 执行方式：按 `P0 / P1 / P2` 三批推进，先补玩法关键缺口，再补内容，再补长线与产品感

---

## 1. 总体判断

当前三个模块都已经不是空壳，但都还没有达到“玩法完整”的标准：

- **吃豆人**：核心单局体验最完整，但内容包和教程练习层不完整
- **祖玛**：单局骨架与道具体系已经成立，但关卡目标系统、后半程内容和模式层不完整
- **三消**：基础盘内循环已成，但高阶特殊块组合、掉落关、模式层和长线内容差距最大

当前建议的整体推进顺序：

1. 祖玛 P0
2. 吃豆人 P0
3. 三消 P0
4. 三消 P1
5. 祖玛 P1
6. 吃豆人 P1
7. 三消 P2
8. 吃豆人 P2
9. 祖玛 P2

## 2. P0：补齐核心玩法缺口

### 2.1 祖玛 P0

#### 目标
- 把祖玛从“前中期可玩原型”补到“完整主玩法成立”

#### 本批必须完成
1. 真实关卡目标系统
   - 限定发射数
   - 连锁次数目标
   - 道具使用目标
   - 多目标组合结算
2. 模式与配置真实生效
   - 切模式 / 关卡包 / 时长时，真实重建开局状态
3. 练习闭环打通
   - 图鉴页开始练习可达
   - 结算页进入练习跳到真实训练入口
4. 对局内退出路径补齐
   - 返回选关 / 返回大厅

#### 验收标准
- 连锁包和挑战关不再是“文案目标”
- 练习页可进入、可完成、可回主玩法
- 切模式后开局参数和 UI 一致
- 对局内不依赖外层工作簿也可退出

---

### 2.2 三消 P0

#### 目标
- 把三消从“基础盘内循环成立”补到“经典三消核心规则完整”

#### 本批必须完成
1. 高阶特殊块组合补齐
   - 彩球 + 特殊块
   - 彩球 + 彩球
   - 条纹 + 包装
   - 包装 + 包装
2. 障碍规则高保真
   - 锁链不可移动
   - 木箱 / 石块按经典邻接受伤
   - 传送入口 / 出口可视区分
3. 掉落收集关正式落地
   - 掉落物、出口、计数、目标、结算
4. 键盘交互重做
   - 焦点移动
   - 选中
   - 确认交换
5. 实验室练习接真模块

#### 验收标准
- 特殊块高阶组合可稳定触发
- 至少一组正式掉落收集关可玩
- 键盘可完整玩一局
- 练习入口不再是假按钮

---
