# 横版自走棋（Auto Chess）Excel 游戏详细设计

- 日期：2026-04-14
- 项目：`excel-aim-trainer`
- 设计目标：在现有 Excel 风格游戏合辑中新增横版自走棋模块，提供"棋子收集 + 羁绊搭配 + 自动战斗"的策略玩法
- 设计阶段：实施前详细设计

---

## 1. 游戏概述

### 1.1 一句话介绍

横版自走棋是一个"买棋子、凑羁绊、自动对战"的策略游戏——在 Excel 表格里，它变成了"数据编排器"，棋子是数据函数，羁绊是公式组合，战斗是数据校验过程。

### 1.2 核心卖点

- **3 秒可理解**：买棋子放棋盘，凑齐羁绊变强，自动打敌人
- **策略深度**：棋子选择、站位安排、经济管理、羁绊搭配，每局都有不同解法
- **随机 + 规划**：商店随机刷新，但需要长远规划阵容
- **Excel 伪装天然适配**：棋盘 = 数据表格，棋子 = 函数单元格，羁绊 = 公式联动，战斗 = 数据校验
- **短局策略循环**：每局 15~25 分钟，每回合 1~2 分钟，随时可做决策

## 2. 原版游戏分析

### 2.1 原版核心机制

| 机制 | 说明 |
| --- | --- |
| 棋盘系统 | 8×8 网格，玩家占据下半部分（4×8），敌人占据上半部分 |
| 棋子购买 | 每回合从随机刷新的商店中购买棋子 |
| 棋子升星 | 3 个相同 1 星棋子合成 2 星，3 个 2 星合成 3 星 |

| 经济系统 | 金币用于购买棋子、刷新商店、购买经验，利息机制鼓励存钱 |
| 连胜/连败 | 连续胜利或失败获得额外金币奖励 |

### 2.2 原版玩法循环

```
回合开始
  -> 准备阶段（30 秒）
  -> 商店刷新，显示 5 个棋子
  -> 购买棋子、调整站位
  -> 升级人口（可选）
  -> 准备阶段结束
  -> 战斗阶段（自动进行）
  -> 棋子自动攻击
  -> 战斗结束，结算伤害
  -> 获得金币奖励
  -> 进入下一回合
```

## 3. 核心玩法设计

### 3.1 玩法规则

#### 基本规则
- 棋盘为 8×4 网格（横版），玩家占据左侧 4×4，敌人占据右侧 4×4
- 每回合准备阶段 30 秒，可购买棋子、调整站位
- 战斗阶段自动进行，棋子自动攻击最近敌人
- 战斗失败扣除 HP，HP 归零则淘汰
- 最后存活的玩家获胜

#### 棋子规则
- 棋子有品质（白/绿/蓝/紫/橙）、职业、种族
- 3 个相同棋子可合成升星（1 星→2 星→3 星）
- 升星后属性大幅提升（约×2.2）
- 棋子占用 1 个人口

#### 羁绊规则
- 凑齐特定数量同职业/种族棋子激活羁绊
- 羁绊效果作用于全局或特定棋子
- 同一棋子可贡献多个羁绊

#### 经济规则
- 每回合基础收入：5 + 回合数
- 利息：每 10 金币获得 1 金币利息，上限 5
- 连胜奖励：3/5/7 连胜获得 1/2/3 额外金币
- 连败奖励：3/5/7 连败获得 1/2/3 额外金币

## 7. 战斗系统设计

### 7.1 战斗流程

```
战斗开始
  -> 棋子进入战斗位置
  -> 寻找最近敌人
  -> 进入攻击状态
  -> 攻击敌人
  -> 积累法力值
  -> 法力值满释放技能
  -> 棋子死亡
  -> 一方全灭，战斗结束
```

### 7.2 攻击逻辑

| 优先级 | 行为 | 说明 |
| --- | --- | --- |
| 1 | 寻找最近敌人 | 计算距离，选择最近目标 |
| 2 | 检查攻击范围 | 在范围内则攻击 |
| 3 | 执行普通攻击 | 造成攻击力伤害 |
| 4 | 积累法力值 | 每次攻击积累 10 法力 |
| 5 | 检查法力值 | 满则释放技能 |
| 6 | 释放技能 | 执行技能效果 |

### 7.3 伤害计算

```ts
// 伤害计算公式
function calculateDamage(attacker: Piece, defender: Piece): number {
  const baseDamage = attacker.attack;
  const defenseReduction = defender.defense / (defender.defense + 100);
  const randomFactor = 0.9 + Math.random() * 0.2; // 0.9 ~ 1.1
  
  let damage = baseDamage * (1 - defenseReduction) * randomFactor;
  
  // 暴击判定
  if (Math.random() < getCritRate(attacker)) {
    damage *= 1.5; // 暴击伤害 1.5 倍
  }
  
  return Math.max(1, Math.floor(damage));
}
```

## 14. 技术难点分析

### 14.1 自动战斗 AI

```ts
// 棋子自动攻击逻辑
class PieceAI {
  update(piece: Piece, enemies: Piece[], allies: Piece[]): void {
    // 寻找最近敌人
    const target = this.findNearestEnemy(piece, enemies);
    
    if (!target) return;
    
    // 检查攻击范围
    const distance = this.calculateDistance(piece, target);
    
    if (distance <= piece.attackRange) {
      // 在范围内，执行攻击
      this.attack(piece, target);
      
      // 积累法力
      piece.skill.currentMana += 10;
      
      // 检查是否释放技能
      if (piece.skill && piece.skill.currentMana >= piece.skill.manaCost) {
        this.castSkill(piece, enemies, allies);
      }
    } else {
      // 不在范围内，移动向敌人
      this.moveTowards(piece, target);
    }
  }
  
  private findNearestEnemy(piece: Piece, enemies: Piece[]): Piece | null {
    let nearest: Piece | null = null;
    let minDistance = Infinity;
    
    for (const enemy of enemies) {
      if (enemy.hp <= 0) continue;
      
      const distance = this.calculateDistance(piece, enemy);
      if (distance < minDistance) {
        minDistance = distance;
        nearest = enemy;
      }
    }
    
    return nearest;
  }
}
```

### 14.2 羁绊计算

```ts
// 羁绊激活计算
function calculateBonds(pieces: Piece[]): Bond[] {
  const roleCount = new Map<PieceRole, number>();
  const raceCount = new Map<PieceRace, number>();
  
  // 统计职业和种族数量
  for (const piece of pieces) {
    if (!piece.position) continue; // 只计算棋盘上的棋子
    
    for (const role of piece.roles) {
      roleCount.set(role, (roleCount.get(role) || 0) + 1);
    }
    for (const race of piece.races) {
      raceCount.set(race, (raceCount.get(race) || 0) + 1);
    }
  }
  
  // 计算激活的羁绊
  const activeBonds: Bond[] = [];
  
  for (const [role, count] of roleCount) {
    const bond = getRoleBond(role, count);
    if (bond) activeBonds.push(bond);
  }
  
  for (const [race, count] of raceCount) {
    const bond = getRaceBond(race, count);
    if (bond) activeBonds.push(bond);
  }
  
  return activeBonds;
}
```

// 商店刷新逻辑
function refreshShop(playerLevel: number, gold: number): Shop {
  const refreshCost = 2;
  if (gold < refreshCost) return null;
  
  const probabilities = getShopProbabilities(playerLevel);
  const pieces: (Piece | null)[] = [];
  
  for (let i = 0; i < 5; i++) {
    const quality = rollQuality(probabilities);
    const piece = generatePiece(quality);
    pieces.push(piece);
  }
  
  return {
    pieces,
    locked: [false, false, false, false, false],
    refreshCost,
  };
}
