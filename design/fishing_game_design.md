# 捕鱼达人（Fishing Master）Excel 游戏详细设计

- 日期：2026-04-14
- 项目：`excel-aim-trainer`
- 设计目标：在现有 Excel 风格游戏合辑中新增捕鱼达人模块，提供"炮台射击 + 鱼类收集 + 金币循环"的经典街机玩法
- 设计阶段：实施前详细设计

---

## 1. 游戏概述

### 1.1 一句话介绍

捕鱼达人是一个"瞄准鱼群开炮，捕获后获得金币，升级炮台继续捕鱼"的休闲游戏——在 Excel 表格里，它变成了"数据猎人"，炮台是数据筛选器，鱼是流动的数据行，金币是数据价值。

### 1.2 核心卖点

- **3 秒可理解**：鼠标瞄准，点击开炮，捕获鱼获得金币
- **风险决策**：小鱼容易抓但金币少，大鱼难抓但金币多，需要权衡
- **炮台升级循环**：用金币升级炮台威力，形成"捕鱼 -> 赚金币 -> 升级 -> 捕更大的鱼"的正循环
- **Excel 伪装天然适配**：炮台 = 数据筛选器，鱼 = 数据行，金币 = 数据价值，渔网 = 批量选择
- **短局高压循环**：每局 3~5 分钟，限时内尽可能多捕鱼

## 3. 核心玩法设计

### 3.1 玩法规则

#### 基本规则
- 玩家控制炮台方向，鼠标瞄准，点击开炮
- 子弹沿直线飞行，击中鱼后计算捕获概率
- 捕获成功获得金币，失败鱼继续游动
- 限时内尽可能多捕鱼，时间结束结算

#### 捕获概率公式
```
捕获概率 = min(炮台威力 / 鱼的生命值 × 基础概率, 最大概率)
```

#### 炮台规则
- 炮台可 360° 旋转，跟随鼠标方向
- 点击发射子弹，子弹消耗金币
- 炮台等级越高，威力越大，子弹消耗越多

#### 鱼类规则
- 鱼从屏幕边缘出现，沿预定路径游动
- 鱼游出屏幕后消失
- 不同鱼有不同生命值、速度和金币奖励

### 12.1 碰撞检测优化

```ts
// 使用空间哈希优化碰撞检测
class SpatialHash {
  private cells: Map<string, Fish[]> = new Map();
  private cellSize: number = 50;

  insert(fish: Fish): void {
    const key = this.getKey(fish.x, fish.y);
    if (!this.cells.has(key)) {
      this.cells.set(key, []);
    }
    this.cells.get(key)!.push(fish);
  }

  query(x: number, y: number, radius: number): Fish[] {
    const results: Fish[] = [];
    const minCell = this.getKey(x - radius, y - radius);
    const maxCell = this.getKey(x + radius, y + radius);
    
    // 遍历相关单元格
    for (const [key, fishes] of this.cells) {
      if (this.isInRange(key, x, y, radius)) {
        for (const fish of fishes) {
          if (this.distance(fish.x, fish.y, x, y) <= radius) {
            results.push(fish);
          }
        }
      }
    }
    
    return results;
  }
}
```

```ts
// 生成鱼类游动路径
function generateFishPath(scene: SceneKind, fish: Fish): PathSegment[] {
  const paths: PathSegment[] = [];
  
  switch (fish.kind) {
    case 'clownfish':
    case 'tropical':
      // 直线游动
      paths.push({
        type: 'linear',
        startX: fish.x,
        startY: fish.y,
        endX: scene.width + 50,
        endY: fish.y,
        duration: 5000,
      });
      break;
      
    case 'dolphin':
      // 正弦曲线游动
      paths.push({
        type: 'sine',
        startX: fish.x,
        startY: fish.y,
        amplitude: 50,
        frequency: 2,
        duration: 8000,
      });
      break;
      
    case 'swordfish':
      // 冲刺
      paths.push({
        type: 'dash',
        startX: fish.x,
        startY: fish.y,
        speed: 200,
        duration: 2000,
      });
      break;
  }
  
  return paths;
}
```

### 12.3 捕获概率计算

```ts
// 计算捕获概率
function calculateCatchProbability(
  cannon: Cannon,
  fish: Fish,
  bullet: Bullet
): number {
  const baseRate = 0.8; // 基础捕获率
  const powerRatio = cannon.damage / fish.hp;
  const maxRate = 0.95; // 最大捕获率
  
  // 应用各种修正
  let rate = Math.min(powerRatio * baseRate, maxRate);
  
  // 冰冻状态提升概率
  if (fish.isFrozen) {
    rate *= 1.5;
  }
  
  // 眩晕状态提升概率
  if (fish.isStunned) {
    rate *= 1.3;
  }
  
  // Boss 鱼降低概率
  if (isBossFish(fish.kind)) {
    rate *= 0.5;
  }
  
  return Math.min(rate, maxRate);
}
```
