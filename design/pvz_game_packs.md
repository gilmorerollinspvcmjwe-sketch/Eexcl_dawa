# PvZ 游戏包设计（Sheet7 / Sheet8 / Sheet9）

## 设计前提
- 参考：`pvz_levels_and_modes_masterplan.md` 的章节/模式矩阵、`pvz_module_analysis.md` 的模块职责、`pvz_reference_redesign.md` 的体验支柱。
- 输出目标：每个包都能拆成“章节配置 + 植物/僵尸规则 + 波次脚本 + HUD/结算 + Sheet 联动”的明确工作项，直接对应 `pvzChapters`、`pvzBoardState`、registry、战斗 UI 与图鉴/实验室入口。
- 风格：保留原版 PvZ 的资源经营、选卡、防线、波次、章节差异，同时压缩成适合 Excel 工作簿短中局的正式内容包。
- 每个包结尾注明“Excel 对应”，便于后续把 `Sheet7/8/9` 的入口、提示与记录统一接起来。

## 游戏包列表

1. **白天征途包：经济与防线入门**  
   - **定位**：作为 PvZ 正式主线的第一层，负责教会玩家“先做经济、再铺输出、按路补防”。  
   - **原版参考**：原版白天 1-1 到 1-10 的教学与旗帜波推进。  
   - **核心规则**：使用 `day` 经济规则、基础植物池、基础僵尸池；每 2~3 波抬一次压力，最后用旗帜波完成收尾；卡组以 `sunflower / peashooter / wallnut / potatoMine` 为核心。  
   - **建议关卡数**：8 关（从单一威胁到多路同时压进）。  
   - **首发必做**：`Sheet9` 章节卡提供目标与推荐卡组；`Sheet7` HUD 显示当前波次、旗帜波提示、最危险路；基础植物能力必须全部完整可用。  
   - **可延期内容**：加入白天小游戏支线、章节内额外勋章目标（如“不损失任何割草机”）。  
   - **Excel 对应**：`章节征途` 默认落在这一包，`pvzChapters` 输出白天 scenario，`Sheet8` 图鉴同步展示本章首次出现的植物与僵尸。  

2. **夜晚经营包：低光资源压缩**  
   - **定位**：把 PvZ 从“正常资源塔防”拉向“紧资源决策”，形成第一轮章节差异。  
   - **原版参考**：原版夜晚章节、蘑菇体系和慢资源节奏。  
   - **核心规则**：使用 `night` 经济规则，基础阳光收入降低；夜间关卡用简化蘑菇/夜间植物替代完整原版体系；僵尸波次更密，要求更早做出防线排序。  
   - **建议关卡数**：6 关（教学夜间经济、夜间快攻、夜间旗帜压迫三段）。  
   - **首发必做**：章节说明页明确“资源减少、默认卡组改变、推荐前期节奏”；`pvzBoardState` 支持夜间经济修正；结算页提示“本章建议先稳住经济还是先抗首波”。  
   - **可延期内容**：加入照明规则、夜雾小事件、完整蘑菇成长链。  
   - **Excel 对应**：`Sheet9` 的夜晚章节卡改变开局阳光、默认植物池和刷怪节奏；`Sheet7` 顶部状态带显示“夜间资源压力”。  

3. **泳池防线包：水陆双线控场**  
   - **定位**：用“部分路段不可按白天逻辑处理”这件事，建立中期章节特色。  
   - **原版参考**：原版泳池章节、潜水僵尸、水生植物布防。  
   - **核心规则**：棋盘中部两路或指定列进入“水域”状态；陆地植物与水生植物的可放置规则不同；潜水类僵尸在接近前线时才重新显形，逼玩家做岸边与水路双重配置。  
   - **建议关卡数**：6 关（先学会水域布防，再处理潜水与岸边双线压力）。  
   - **首发必做**：scenario 里加入 `boardModifiers` 水域标记；放置判定识别水陆差异；HUD/公式栏提示“本关水路威胁”和“可用水生位”。  
   - **可延期内容**：加入更复杂的潜水计时、泳池清理挑战和水域专属精英波。  
   - **Excel 对应**：`Sheet7` 棋盘直接渲染水域格；`Sheet9` 章节包切换后，`pvzBoardState` 读取不同地形与僵尸脚本；`Sheet8` 增加水生植物和潜水僵尸说明。  

4. **迷雾侦察包：视野限制与预判防守**  
   - **定位**：把 PvZ 的“局内信息不完整”做出来，让玩家依赖阵型、预判和侦察位。  
   - **原版参考**：原版迷雾章节与视野受限的推进方式。  
   - **核心规则**：部分列被迷雾遮蔽，玩家不能持续完整看到来袭信息；通过探照型植物、章节提示和短时揭示机制维持判断；波次压力转向“未知来袭 + 多路修补”。  
   - **建议关卡数**：5 关（从轻雾到浓雾，再到旗帜波浓雾爆发）。  
   - **首发必做**：UI 上让迷雾是真正的可视限制而不是文案描述；`Sheet7` 显示“侦察覆盖不足”的提示；结算页复盘哪几路在迷雾中失守。  
   - **可延期内容**：加入移动雾墙、夜雾混合章节、实验室自定义迷雾规则。  
   - **Excel 对应**：`Sheet7` 主棋盘对指定格做遮蔽层；`Sheet9` 的章节说明中突出“本章推荐侦察/延迟型卡组”；`Sheet8` 图鉴补充迷雾相关单位标签。  

5. **屋顶反应包：抛投与高压快节奏**  
   - **定位**：作为高阶主线章节，强调“地形变化 + 高速判断 + 特定植物价值”。  
   - **原版参考**：原版屋顶章节、斜坡抛投感和高空僵尸压力。  
   - **核心规则**：指定区域作为屋顶斜面或不可直接种植区，玩家要通过抛投植物、保护核心路和快速补位维持防线；波次会更频繁地混入高速僵尸。  
   - **建议关卡数**：6 关（斜坡教学、快攻、混编精英、最终多路旗帜关）。  
   - **首发必做**：scenario 能定义“特殊放置限制”；卡组推荐明确哪些植物适配屋顶；HUD 持续提示哪一路是当前突破口。  
   - **可延期内容**：加入更复杂的平台事件、弹射植物组合和屋顶专属无尽。  
   - **Excel 对应**：`Sheet9` 屋顶包改写默认卡组与板面限制；`Sheet7` 用地形样式强调屋顶列；`Sheet8` 图鉴为抛投/高空应对单位做专题说明。  

6. **洞察试验包：小游戏与规则实验**  
   - **定位**：把原版小游戏、解谜、特殊规则关压缩成适合本项目的挑战集合，而不是新开一套复杂玩法框架。  
   - **原版参考**：原版小游戏、解谜模式、限定植物与特殊任务关。  
   - **核心规则**：沿用主战斗 simulation，但每关附加一条硬规则，如“禁用向日葵”“限定 4 卡槽”“只允许爆发植物”“规定开局阵型”；目标不再只是通关，还可能是指定波次内完成任务。  
   - **建议关卡数**：10 关首发，按“教学试验 / 资源试验 / 阵型试验 / 限定通关”分组。  
   - **首发必做**：`Sheet9` 的 `洞察试验` 入口变成真实可选的挑战包列表；每关有规则摘要、推荐解法方向和完成记录；结算页能显示本次试验目标是否达成。  
   - **可延期内容**：加入解谜脚本工具、实验室自定义规则组合器、每日试验。  
   - **Excel 对应**：`Sheet9` 直接承担试验包选择与记录；`Sheet7` 复用战斗页，只额外挂载规则条；`Sheet8` 补“试验重点单位”说明。  

7. **长线生存包：高压连续守线**  
   - **定位**：提供给熟练玩家的长期投入内容，承接原版 Survival / Endless 的反复挑战体验。  
   - **原版参考**：原版生存模式与无尽模式的波段强化。  
   - **核心规则**：按“阶段”而不是单章结束；每阶段提高僵尸混编强度、缩短准备窗口，并允许玩家在阶段间有限调整卡组或阵型；目标是尽可能撑更久而不是只打完固定终点。  
   - **建议关卡数**：3 个生存主题包（白天生存、夜晚生存、章节混合生存），每个包内含无尽阶段。  
   - **首发必做**：生存模式专属 HUD，显示当前阶段、历史最远阶段、主要压力源；记录层保存最佳阶段、常用卡组和最近失败原因。  
   - **可延期内容**：加入日夜轮换、特殊事件阶段、排行榜和周挑战。  
   - **Excel 对应**：`长线生存` 入口挂在 `Sheet7/9` 现有模式结构下；`pvzBoardState` 读取循环阶段脚本；结算 overlay 显示“撑到第几阶段”和“推荐下次替换卡组”。  

以上每个包都能直接拆成实施任务（章节脚本、地形修饰、植物规则、僵尸组合、HUD/结算、进度记录分别一组），后续开发只需围绕 `pvzChapters`、`pvzBoardState`、单位 registry、`PvZGameSheet` 和 `Sheet8/9` 回流入口逐包推进，就能把 PvZ 从“可玩第一版”升级成可持续更新的正式章节模块。
