## 🎮 模式详细设计

### 1. 静态定位模式（Static Targeting）

#### 核心机制

- 靶子随机在单元格生成（B2-AD50区域）
- 点击命中后立即消失
- 立即刷新下一个靶子
- 可配置同时存在的靶子数量

#### 判定规则

| 判定要素 | 规则 |
|----------|------|
| 命中区域 | 整个单元格 |
| 有效点击 | 鼠标点击在目标单元格内 |
| 超时处理 | 靶子持续时间到期后消失，不计miss |

#### 参数配置

| 参数名称 | 范围 | 默认值 | 说明 |
|----------|------|--------|------|
| 靶子持续时间 | 1-10秒 | 3秒 | 目标存活时间 |
| 刷新间隔 | 0.5-3秒 | 1.5秒 | 目标生成间隔 |
| 同时靶子数 | 1-5 | 2 | 场上可存在目标数 |
| 靶子大小 | 50%-150% | 100% | 目标显示大小 |

#### 计分方式

| 部位 | 基础分 | 说明 |
|------|--------|------|
| 头部 | 100分 | 最佳目标 |
| 身体 | 50分 | 中等得分 |
| 脚部 | 25分 | 最低得分 |

| 连击数 | 倍率 |
|--------|------|
| 0-4 | 1.0x |
| 5-9 | 1.5x |
| 10-19 | 2.0x |
| 20-29 | 2.5x |
| 30+ | 3.0x |

#### 摸鱼安全等级：★★★★★

---

### 2. 爆头线专项模式（Headshot Line）

#### 核心机制

- 所有靶子的头部固定在同一行（可配置行号，默认第10行）
- 仅头部命中计分，身体/脚部命中无效
- 强制训练爆头肌肉记忆
- 适合瓦罗兰特/CS2 玩家

#### 判定规则

| 判定要素 | 规则 |
|----------|------|
| 头部位置 | 固定行（如第10行） |
| 有效命中 | 仅头部单元格点击有效 |
| 身体/脚部 | 点击不计分，也不重置连击 |
| 超时 | 超时后目标消失，不计miss |

#### 参数配置

| 参数名称 | 范围 | 默认值 | 说明 |
|----------|------|--------|------|
| 爆头线行号 | 5-45 | 10 | 头部出现的固定行 |
| 目标持续时间 | 1-10秒 | 2.5秒 | 相比静态模式更短 |
| 生成间隔 | 0.5-3秒 | 1秒 | 更快的刷新节奏 |
| 同时目标数 | 1-3 | 1 | 减少干扰，专注爆头 |

#### 计分方式

| 命中类型 | 得分 | 说明 |
|----------|------|------|
| 头部命中 | 150分 | 高于其他模式的头部分数 |
| 身体/脚部 | 0分 | 不计分也不重置连击 |

**连击倍率**：与静态模式相同

#### 摸鱼安全等级：★★★★★

---

### 3. 动态跟枪模式（Tracking）

#### 核心机制

- 靶子在单元格内匀速平移或斜向移动
- 玩家需要跟住移动的目标点击命中
- 模拟FPS游戏中的移动靶跟枪

#### 判定规则

| 判定要素 | 规则 |
|----------|------|
| 移动范围 | 单元格内移动，不跨单元格 |
| 有效命中 | 鼠标点击到移动中的目标 |
| 移动类型 | 水平/垂直/斜向/随机折线 |
| 判定时机 | 点击瞬间目标位置 |

#### 参数配置

| 参数名称 | 范围 | 默认值 | 说明 |
|----------|------|--------|------|
| 移动速度 | 慢/中/快/极快 | 中 | 目标移动速度 |
| 移动方向 | 水平/垂直/斜向/随机 | 随机 | 移动轨迹类型 |
| 目标持续时间 | 2-10秒 | 5秒 | 移动目标持续时间 |
| 同时目标数 | 1-3 | 1 | 减少干扰 |

**速度对照**：

| 等级 | 移动速度 | 适用场景 |
|------|----------|----------|
| 慢 | 50px/s | 新手入门 |
| 中 | 100px/s | 基础跟枪 |
| 快 | 150px/s | 进阶训练 |
| 极快 | 200px/s | 高手挑战 |

#### 计分方式

| 命中时机 | 得分加成 |
|----------|----------|
| 常规命中 | 1.0x 基础分 |
| 移动中命中 | 1.2x 基础分 |
| 精准命中（目标中心） | 1.5x 基础分 |

**部位得分**：与静态模式相同

#### 摸鱼安全等级：★★★★☆

---

### 4. 急停定位模式（Peek Stop）

#### 核心机制

- 方向键（←/→）控制靶子横向平移，模拟人物移动
- 空格键触发急停
- 仅急停后 0.5 秒内的命中有效
- 移动中命中不计分

#### 判定规则

| 判定要素 | 规则 |
|----------|------|
| 移动控制 | 方向键←→控制横向移动 |
| 急停触发 | 空格键 |
| 有效窗口 | 急停后 0.5 秒 |
| 移动中点击 | 无效，不计分也不重置连击 |
| 超时处理 | 目标到达边界或超时后重置位置 |

#### 参数配置

| 参数名称 | 范围 | 默认值 | 说明 |
|----------|------|--------|------|
| 移动速度 | 慢/中/快 | 中 | 靶子横向移动速度 |
| 有效窗口 | 0.3-1.0秒 | 0.5秒 | 急停后的有效命中时间 |
| 目标持续时间 | 3-10秒 | 5秒 | 单个目标存活时间 |
| 移动范围 | 3-10列 | 5列 | 横向移动范围 |

**速度对照**：

| 等级 | 移动速度 | 说明 |
|------|----------|------|
| 慢 | 80px/s | 适合新手 |
| 中 | 120px/s | 标准训练 |
| 快 | 180px/s | 进阶挑战 |

#### 计分方式

| 命中时机 | 得分 |
|----------|------|
| 急停窗口内命中 | 200分 |
| 移动中命中 | 0分 |

**连击机制**：急停命中后连击继续，miss重置

#### 操作说明

```
[←] 按住：靶子向左移动
[→] 按住：靶子向右移动
[空格] 点击：触发急停 → 进入0.5秒有效窗口
```

#### 摸鱼安全等级：★★★★☆

---

### 5. peek闪镜模式（Flash Peek）

#### 核心机制

- 靶子从表格左右两侧快速滑入
- 在表格区域停留 0.3-0.8 秒
- 停留窗口内命中有效，过时不计分

#### 判定规则

| 判定要素 | 规则 |
|----------|------|
| 进入方向 | 左侧或右侧随机 |
| 停留时间 | 0.3-0.8秒（随机） |
| 有效区域 | 表格中间区域（B-AD列） |
| 滑出方向 | 从另一侧滑出 |

#### 参数配置

| 参数名称 | 范围 | 默认值 | 说明 |
|----------|------|--------|------|
| 滑入速度 | 慢/中/快 | 中 | 靶子滑入速度 |
| 停留时间 | 0.3-1.0秒 | 0.5秒 | 有效命中窗口 |
| 滑出速度 | 与滑入相同 | - | 离开速度 |
| 目标间隔 | 1-5秒 | 2秒 | 下一个目标出现间隔 |

#### 计分方式

| 命中时机 | 得分 |
|----------|------|
| 停留窗口内命中 | 150分 |
| 滑入/滑出时命中 | 0分 |

**连击机制**：同静态模式

#### 摸鱼安全等级：★★★☆☆

```
│  │ 综合系数:    │  1.0         │ ← 可调 0.1-5.0               │

│  │ Y轴系数:     │  1.0         │ ← 可调 0.1-5.0               │
│  └──────────────┴──────────────┘                                │
│                                                                  │
│  [应用瓦罗兰特预设] [应用CS2预设] [应用CF预设] [自定义]         │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 预设游戏换算模板

| 游戏 | DPI | 游戏灵敏度 | 推荐系数 |
|------|-----|------------|----------|
| 瓦罗兰特 | 800 | 0.40 | 1.0 |
| CS2 | 800 | 1.0 | 2.5 |
| CF | 1000 | 30 | 75 |
| APEX | 800 | 2.2 | 5.5 |

### 计算公式

```
Excel练枪移动距离 = 实际鼠标移动距离 × 灵敏度系数 × (游戏灵敏度换算系数)
```

### XY轴分离调节

- **综合系数**：同时影响 X 轴和 Y 轴
- **X轴系数**：单独调节水平方向灵敏度
- **Y轴系数**：单独调节垂直方向灵敏度
