# Excel 风格摸鱼游戏合辑中心设计稿

- 日期：2026-04-11
- 项目：`excel-aim-trainer`
- 设计范围：首页架构、共享系统、五个游戏模块、风格约束、功能总表
- 当前阶段：产品/交互设计确认稿

---

## 1. 设计结论摘要

本项目将从当前的 **Excel 伪装练枪游戏**，升级为一个 **Excel 风格摸鱼游戏合辑中心**。

已经确认的方向如下：

- 总体气质：**半伪装半整活型**
- 产品结构：**共享成长型**
- 游戏关系：**练枪 / 贪吃蛇 / 俄罗斯方块 / 拼豆 / 植物大战僵尸平级**
- 命名包装：**轻包装**
  - 保留原游戏名
  - 主要通过 Excel 外壳、文案、表格布局、状态文案来实现统一气质
- UI 原则：**延续当前项目的 Excel 风格，不改成现代游戏大厅**
- 拼豆定义：**像素拼豆 / 热熔拼豆式治愈创作玩法**

该版本不追求把所有游戏做成同一套路，而是把它们统一到同一个“工作簿宇宙”里：

- 练枪 = 爽感 / 热手 / 爆发
- 贪吃蛇 = 上头 / 短局循环
- 俄罗斯方块 = 整理 / 心流 / 稳定刷分
- 拼豆 = 治愈 / 收集 / 创作
- 植物大战僵尸 = 守线 / 策略 / 推关

---

## 2. 产品目标

### 2.1 一句话定位

一个看起来像 Excel 工作簿、实际上是工位摸鱼游戏合辑的共享成长中心。

### 2.2 核心体验目标

1. 第一眼仍然像当前这个项目，而不是新起一套 UI。
2. 首页从“练枪入口”升级为“合辑总控台”。
3. 五个游戏都有各自鲜明节奏，不互相抢位。
4. 所有游戏共享等级、货币、称号、主题、图鉴、隐藏内容。
5. 用户既能碎片化开一局，也能长期解锁与收集。

### 2.3 不做什么

- 不改成现代网页式卡片大厅
- 不给每个游戏单独做一套完全不同的 UI 语言
- 不把全部游戏强行塞进同一种玩法模板
- 不重包装到失去原游戏辨识度

## 3. 风格总原则

### 3.1 必须保留的视觉骨架

- Excel 标题栏
- 菜单栏 / Ribbon
- 公式栏
- Sheet Tabs
- 状态栏
- 单元格式布局
- 经典绿色为主的工作簿气质

### 3.2 允许强化的表现

- 更完整的报表式分区
- 更明确的行列表头
- 游戏化状态文案
- 条件格式感高亮
- 轻量发光、闪烁、进度变化
- 带一点办公室怪味的系统通知

### 3.3 创意来源

抽象感不主要靠夸张立绘或换皮，而主要来自：

- 文件名
- 公式栏吐槽
- 任务文案
- 系统通知
- 隐藏工作表
- 老板巡检 / 周会 / 审计等事件

## 4. 总体信息架构

项目整体推荐采用如下工作表结构：

| Sheet | 名称 | 作用 | 说明 |
| --- | --- | --- | --- |
| Sheet1 | 游戏中心 | 首页总控台 | 游戏入口、继续游玩、任务、成长、动态 |
| Sheet2 | 游戏区 | 当前游戏容器 | 所有游戏共用壳，仅替换中间主内容 |
| Sheet3 | 统计 | 历史成绩与周报 | 各游戏统计、趋势、最高纪录 |
| Sheet4 | 成长 | 员工等级/称号/货币/解锁 | 长期目标与奖励汇总 |
| Sheet5 | 图鉴 | 拼豆作品、隐藏模板、主题收藏 | 收集向内容中心 |
| Sheet6 | 设置 | 全局设置 | 声音、视觉、皮肤、隐藏模式等 |

当前已有结构：

- `hub`
- `game`
- `stats`
- `settings`

设计上建议继续沿用现有路由/状态切换方式，只把 `hub` 扩展成真正的首页总控表，并补充 `growth` / `collection` 两类工作表。

## 5. 首页（游戏中心）架构

### 5.1 首页职责

首页不是传统“游戏大厅”，而是 Excel 风格的总控工作表，承担以下职责：

1. 进入任意游戏
2. 继续上次游玩
3. 查看今日任务
4. 查看共享成长数据
5. 查看近期解锁 / 通知 / 彩蛋

### 5.2 首页推荐版式

#### 顶部区

- 工作簿标题
- 一句系统文案
- 继续游玩入口
- 今日推荐入口

#### 主体左侧

- 五个游戏构成的 **目录总表**
- 一行一个游戏
- 重点不是大图，而是可读、可扫、可快速启动

#### 主体右侧

- 今日任务
- 等级 / 称号 / 摸鱼币 / 连登

#### 底部区

- 最近动态
- 最近解锁
- 快捷导航（统计 / 成长 / 图鉴 / 设置）

### 5.3 首页目录总表（推荐）

| 游戏 | 状态 | 最佳记录 | 今日次数 | 快速开始 |
| --- | --- | ---: | ---: | --- |
| 练枪 | 建议热手 | 最高分 / 最高连击 | N | 启动 |
| 贪吃蛇 | 数据流通畅 | 最长长度 | N | 启动 |
| 俄罗斯方块 | 堆积待整理 | 最高消行 | N | 启动 |
| 拼豆 | 创作中 | 当前模板完成度 | N | 继续/启动 |
| 植物大战僵尸 | 防线待检查 | 最高关卡 | N | 启动 |

### 5.4 首页交互规则

| 操作 | 行为 |
| --- | --- |
| 单击游戏行 | 选中该游戏，右侧显示摘要，公式栏显示介绍 |
| 双击游戏行 | 直接进入对应游戏 |
| 点击“启动/继续” | 直接进入对应游戏 |
| 悬浮任务 | 公式栏显示奖励说明 |
| 悬浮动态 | 公式栏显示详细提示 |

### 5.5 首页系统文案风格

示例：

- `今日建议：先热身，再摸鱼，再伪装。`
- `工作簿状态：稳定运行中。`
- `系统提示：检测到娱乐效率略高于工作效率。`
- `审计建议：请均衡游玩各模块。`

## 6. 共享成长系统

共享成长是该合辑成立的核心。所有游戏共享长期反馈，但各自产出侧重不同。

### 6.1 共享资源总表

| 系统 | 作用 | 产出来源 | 消耗/用途 |
| --- | --- | --- | --- |
| 员工等级 EXP | 长期成长主线 | 所有游戏 | 提升等级、解锁内容 |
| 摸鱼币 | 通用货币 | 所有游戏、任务、成就 | 解锁主题、皮肤、文案包、隐藏内容 |
| 员工称号 | 身份标签 | 成就/里程碑 | 展示在首页、结算页、成长页 |
| 成就 | 长线目标 | 跨游戏完成条件 | 奖励 EXP / 摸鱼币 / 特殊解锁 |
| 图鉴/收藏 | 收集驱动 | 拼豆、主题、隐藏模板、特殊事件 | 首页展示、收藏页浏览 |
| 主题皮肤 | 视觉长期反馈 | 解锁获得 | 改变工作簿皮肤、边框、状态色 |

### 6.2 等级系统

| 等级元素 | 设计建议 |
| --- | --- |
| 显示位置 | 首页成长区、状态栏、结算页 |
| 增长来源 | 任意游戏完成一局、任务、成就 |
| 节奏 | 高频小增长，阶段性解锁明显奖励 |
| 解锁内容 | 新主题、特殊文案、隐藏工作表、头像/徽章、游戏彩蛋 |

### 6.3 摸鱼币用途

| 用途 | 示例 |
| --- | --- |
| 主题皮肤 | 经典绿、夜班蓝、审计紫、警报红 |
| UI 装饰 | 状态栏样式、单元格边框、公式栏皮肤 |
| 特效包 | 练枪反馈、贪吃蛇轨迹、俄罗斯方块消行效果 |
| 文案包 | 系统吐槽语料、事件提示风格 |
| 收藏内容 | 图鉴展示框、拼豆模板包 |
| 隐藏内容 | 隐藏 Sheet、彩蛋模式入口 |

### 6.4 成就体系方向

| 类型 | 示例 |
| --- | --- |
| 新手成就 | 完成任意一局 |
| 跨游戏成就 | 同一天体验 3 个不同游戏 |
| 技术成就 | 练枪 20 连击 / 俄罗斯方块 4 连消 |
| 收集成就 | 拼豆完成 10 个模板 |
| 长线成就 | 连续登录 7 天 |
| 彩蛋成就 | 发现隐藏工作表 |

### 6.5 员工称号示例

- 单元格清道夫
- 数据猎人
- 工位蛇王
- 消行主任
- 拼豆文员
- 防线管理员
- 宏命令大师
- 摸鱼总监

## 9. 游戏详细功能总表

## 9.1 练枪

### 定位

合辑中的高爆发竞技模块，维持当前项目的王牌地位，但不再是唯一主模式。

### 功能表

| 类别 | 设计 |
| --- | --- |
| 首页状态 | 建议热手 / 近期手感 / 今日局数 |
| 基础模式 | 限时、无尽、生存、爆头、FPS 专项 |
| 核心循环 | 进入 -> 快速击破 -> 获取评分 -> 回首页 |
| 额外机制 | 优先级目标、异常单元格、Boss 错误值 |
| 创意事件 | 老板靠近、周会干扰、网络波动 |
| 局内播报 | 命中反馈、连击提示、异常修复播报 |
| 结算指标 | 命中率、爆头率、峰值连击、反应速度 |
| 奖励侧重 | 高额 EXP、竞技称号、特效解锁 |
| 风格气质 | 冷酷、利落、异常清理 |

### 练枪创意强化点

- 敌人/靶子可轻度包装为异常数据块
- 公式栏负责实时播报和吐槽
- 结算页做成“绩效训练报告”
- 后期加入 `#REF!` / `#DIV/0!` 等 Boss 异常

---

## 9.2 贪吃蛇

### 定位

高频短局模块，承担“再来一把”的循环吸引力。

### 功能表

| 类别 | 设计 |
| --- | --- |
| 首页状态 | 数据流通畅 / 最长数据流 / 今日局数 |
| 基础模式 | 经典、限时、无尽、干扰模式 |
| 核心循环 | 移动 -> 吞噬 -> 增长 -> 失误死亡 -> 立刻重开 |
| 食物类型 | 普通数字、摸鱼币、咖啡、金色单元格、错误值 |
| 场地机制 | 冻结窗格、筛选区、合并单元格、动态障碍 |
| 创意事件 | 会议邀请遮挡、路径残影、数据洪流 |
| 结算指标 | 最长长度、存活时间、特殊食物数、金币数 |
| 奖励侧重 | 连续游玩奖励、蛇主题皮肤、路径成就 |
| 风格气质 | 狡猾、贪婪、越来越乱 |

### 贪吃蛇创意强化点

- 蛇可被定义为“数据流 / 宏指令链”
- 特殊食物制造明显风险收益
- 地图变化要有 Excel 逻辑感，而不只是随机障碍

---

## 9.3 俄罗斯方块

### 定位

稳定中时长模块，负责提供刷分心流与“整理混乱”的爽感。

### 功能表

| 类别 | 设计 |
| --- | --- |
| 首页状态 | 堆积待整理 / 最高消行 / 今日局数 |
| 基础模式 | 经典、加班、审计、挑战 |
| 核心循环 | 下落 -> 摆放 -> 消行 -> 加速 -> 崩盘或高分 |
| 视觉单位 | 单元格方块、格式块、浅数字块 |
| 特殊块 | SUM、IF、VLOOKUP、#N/A、Pivot |
| 创意事件 | 领导插需求、垃圾行、压力预警 |
| 结算指标 | 消行数、连消数、清板次数、生存时间 |
| 奖励侧重 | 极简主题、整理类称号、结构成就 |
| 风格气质 | 秩序感、整理欲、压迫下维持整齐 |

### 俄罗斯方块创意强化点

- 消行动画应像“整行被整理并归档”
- 顶部压力越大，公式栏吐槽越阴间
- 该模块适合最纯粹的分数竞争

---

## 9.4 拼豆

### 定位

治愈创作与图鉴收集核心，是合辑里最慢、最安静、最个人化的模块。

### 功能表

| 类别 | 设计 |
| --- | --- |
| 首页状态 | 创作中 / 当前模板进度 / 作品总数 |
| 基础模式 | 模板创作、自由拼图、图鉴浏览 |
| 核心循环 | 选模板 -> 填豆 -> 完成 -> 定型 -> 收藏 |
| 工具系统 | 单点、吸色、填充、对称辅助、撤销重做 |
| 模板来源 | 表情包、办公用品、游戏联动模板、隐藏模板 |
| 创意事件 | 灵感时刻、颜色奖励、隐藏模板出现 |
| 结算指标 | 完成度、耗时、精确度、作品评级 |
| 奖励侧重 | 图鉴、贴纸、主页装饰、个性化主题 |
| 风格气质 | 温和、解压、反差萌 |

### 拼豆创意强化点

- 每个单元格就是天然拼豆位
- 完成后的“热熔定型”流程要有仪式感
- 自由创作内容可反哺首页装饰和徽章系统

---

## 9.5 植物大战僵尸

### 定位

中局运营与关卡推进主力，是合辑里内容池最深的策略模块。

### 功能表

| 类别 | 设计 |
| --- | --- |
| 首页状态 | 防线待检查 / 最高关卡 / 星级记录 |
| 基础模式 | 关卡推进、防线挑战、事件关 |
| 核心循环 | 种植 -> 守线 -> 攒资源 -> 过波 -> 通关 |
| Excel 机制 | 冻结窗格列、筛选行、合并格、大型单位位 |
| 创意单位 | IF 植物、SUM 植物、VLOOKUP 植物、Pivot 植物 |
| 创意事件 | 午休、周会、停电、年终审计、空调过冷 |
| 结算指标 | 关卡完成度、星级、资源效率、漏怪情况 |
| 奖励侧重 | 稀有主题、高级称号、隐藏关卡 |
| 风格气质 | 防线管理、秩序保卫、持续压力 |

### 植物大战僵尸创意强化点

- 网格与原作契合极强，应重点做行列机制创新
- 办公室皮肤可做，但不需要强改原名
- 这是最适合承载“长期推关”的模块

## 11. 统一事件与彩蛋系统

### 11.1 通用事件

| 事件 | 影响 |
| --- | --- |
| 老板靠近 | 提示增强、界面伪装压力提升 |
| 周会开始 | 某些游戏出现额外干扰 |
| 审计抽查 | 首页动态更新、特定挑战出现 |
| 加班时段 | 夜间主题、节奏更紧张 |
| 异常工作表闪烁 | 提示隐藏内容 |

### 11.2 公式栏吐槽系统

公式栏应成为统一的“文本戏剧层”，示例：

- 练枪：`=你今天的手感有点离谱()`
- 贪吃蛇：`=IF(继续贪, 继续长, 迟早死)`
- 俄罗斯方块：`=压力值^2`
- 拼豆：`=今日心情 + 20%`
- 植物大战僵尸：`=防线完整性 - 你现在的慌张`

### 11.3 隐藏内容方向

- `SheetX`
- `DONOTOPEN`
- `绩效复核`
- `豆豆图鉴`
- `异常日志`

隐藏内容建议通过：

- 成就
- 深夜登录
- 跨游戏条件
- 特定连击 / 连胜 / 完成率

来逐步解锁。

## 13. 推荐落地顺序

### 13.1 内容优先级

1. 首页升级为合辑总控台
2. 练枪转为平级游戏模块
3. 接入贪吃蛇
4. 接入俄罗斯方块
5. 接入拼豆
6. 接入植物大战僵尸
7. 完善成长、图鉴、主题、隐藏内容

### 13.2 原因

- 首页一旦改完，合辑身份就成立
- 练枪可直接复用，是现成内容
- 贪吃蛇 / 俄罗斯方块能快速建立“不是单游戏”的认知
- 拼豆带来节奏反差与图鉴系统价值
- 植物大战僵尸最重，适合作为中后期内容池

## 14. 验收标准（设计层）

如果后续实现符合以下标准，则说明本设计方向成立：

1. 首页一眼看出是 Excel 风格的游戏总控台
2. 五个游戏入口平级且清晰
3. 不同游戏进入后仍明显处于同一套工作簿系统里
4. 玩家能从任何游戏获得共享成长反馈
5. 拼豆与练枪的气质差异明显但不割裂
6. 公式栏、状态栏、系统文案成为统一氛围层
7. 合辑不是“拼盘”，而像一个有生命的工位娱乐系统

## 16. 最终设计判断

该项目最独特的地方，不是“Excel 皮小游戏”，而是：

**把不同摸鱼情绪，统一放进同一个 Excel 工作簿宇宙里。**

它不是纯怀旧，不是纯恶搞，也不是简单合集，而是一套有持续成长、有情绪节奏分工、有办公室幽默感的桌面游戏系统。

如果实现得当，它会具备以下记忆点：

- 外壳极统一
- 内部玩法差异大
- 文案与系统很会整活
- 收藏和成长让人愿意一直开这个“工作簿”
