## 🐛 Bug 列表

### P0 - 阻断性问题

| Bug ID | 问题描述 | 位置 | 修复建议 |
|--------|----------|------|----------|
| P0-1 | 音效设置无效 | `ExcelGrid.tsx` | `playHitSound` 和 `playMissSound` 未检查 `settings.soundEnabled`，需在调用前添加判断 |
| P0-2 | 统计数据累积计算错误 | `useGameLogic.ts` | `endGame` 中计算 `avgScore/accuracy/headAccuracy` 使用错误的加权平均，应存储总场次后重新计算 |
| P0-3 | 命中特效未渲染 | `ExcelGrid.tsx` | `hitEffects` 定义了但未在 JSX 中使用，需添加渲染逻辑 |
| P0-4 | 随机数泄露隐藏状态 | `ExcelGrid.tsx` | 隐藏状态返回的表格显示 `Math.random() > 0.92 ? Math.floor(Math.random() * 1000) : ''`，应返回真正空白或历史数据 |

### P1 - 重要问题

| Bug ID | 问题描述 | 位置 | 修复建议 |
|--------|----------|------|----------|

| P1-2 | 游戏无暂停 | `useGameLogic.ts` | 缺少 `togglePause` 功能，建议添加 P 键或右键菜单暂停 |
| P1-3 | 爆头线行高亮不明显 | `ExcelGrid.tsx` | 仅第一列有箭头指示，应在整个行添加红色背景线 |
| P1-4 | 游戏时长未生效 | `SettingsPanel.tsx` + `useGameLogic.ts` | 传递给 `onStartGame` 的 duration 参数未在 `startGame` 中使用 |

### P2 - 轻微问题

| Bug ID | 问题描述 | 位置 | 修复建议 |
|--------|----------|------|----------|
| P2-1 | 准星光标 SVG 编码问题 | `App.tsx` | URL 编码的 SVG 可能在某些浏览器显示异常，建议使用 base64 或外部文件 |

## 💻 代码质量评估

### TypeScript 类型问题

| 问题 | 位置 | 建议 |
|------|------|------|
| `window as any` | `ExcelGrid.tsx` | 使用 `window.webkitAudioContext` 类型声明 |
| `gameState.mode` 使用 `string` | `ExcelGrid.tsx` | 应使用 `GameMode` 类型 |
| `selectedCell` 类型冗余 | 多处 | 定义 `CellPosition` 类型但未统一使用 |

### 组件结构

**优点**：
- 职责分离清晰（ExcelGrid/ExcelHeader/SheetTabs 等）
- 使用自定义 hooks (useGameLogic) 分离游戏逻辑
- CSS 模块化（index.css）

**问题**：
- hitEffects 未使用但定义了
- ExcelGrid 组件过于庞大（400+ 行）
- 部分样式应迁移到 Tailwind 类

### 性能问题

| 问题 | 位置 | 影响 |
|------|------|------|
| 每 100ms 清理检查 | `useGameLogic.ts` | 可能导致不必要的重渲染 |
| localStorage 同步写入 | `useGameLogic.ts` | 频繁写入可能阻塞 UI |
| CSS 动画 | `index.css` | 可以使用 `transform` 优化 |
